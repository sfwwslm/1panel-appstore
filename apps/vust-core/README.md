# Vust Core

为 [Vust Desk](https://github.com/sfwwslm/vust-desk) 应用提供后端支持的核心服务。它负责处理用户认证、数据同步以及其他关键的业务逻辑。

## ✨ 主要功能

* **用户认证**:
  * 提供安全的**注册**和**登录**功能。
  * 使用 **JWT (JSON Web Tokens)** 进行会话管理和 API 认证。
* **前端界面**:
  * 一个独立的、科技感风格的前端应用，用于用户注册和登录。
  * 登录后提供一个基础的**仪表盘 (Dashboard)** 页面。
* **数据持久化**:
  * 使用 **SQLx** 与数据库进行异步、安全的交互。
  * 通过 `.env` 文件轻松配置数据库连接。
